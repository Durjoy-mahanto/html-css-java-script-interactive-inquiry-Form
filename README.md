# Interactive Inquiry Form

A modern, responsive **Interactive Inquiry Form** built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. The project features a clean split-layout design with an attractive orange gradient information panel and a stylish inquiry form suitable for business, portfolio, agency, or service websites.
